Pong

Date: March 3, 2021

Language: C++

This project is the second in a series of six mini games that was created for my Game Programming class in Spring 2021. It was implemented using C++ and an OpenGL library. This is a simple game of Pong for two players. The keyboard is used to control the different paddles. When the coll collides with the paddles or wall, the ball will change direcction. The figures used for the ball and paddles was created by me using Adobe Photoshop. To download or view the project, click the link in the title.

Sunny Day Scene

Date: February 17, 2021

Language: C++

This project is the first in a series of six mini games that was created for my Game Programming class in Spring 2021. It was implemented using C++ and an OpenGL library. My goal was to create a simple game scene, that when run, depicts a sunny, partly cloudy sky. The program transforms the sun, making it appear as if it is rotating in place, as well as shrinking and growing. Transformations are also being done on the clouds as they are translated across the screen. All art is original, created using Photoshop. To download or view the project, click the link in the title.
